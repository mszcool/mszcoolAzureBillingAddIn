Welcome to the Azure Billing Add-In!

This add-in is supposed to help you with creating Excel-based Azure pricing calculations. For retrieving the latest prices, the add-in makes use of the Azure RateCard APIs!
This means for using the add-in you need to perform the following steps:
  •  Select your subscription
  •  Select a region
  •  Choose your resource type
  •  Insert data into Excel
Have much fun!
